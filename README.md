## Como rodar localmente

**Pré-requisitos:** Node.js (>=16), npm ou Yarn

```bash
# instalar dependências
npm install
# ou
yarn

# rodar em modo desenvolvimento (Vite)
npm run dev
# ou (Create React App)
npm start

# build de produção
npm run build

# checar build localmente (Vite)
npm run preview
```
